# 缩放五连杆机构等效性证明与虚拟模型控制 (VMC) 力矩映射推导

在双轮足等复杂机器人的底层轻量化控制框架中，静力学映射尤为关键。它能够将内环独立计算的腿长支撑力（如基于重力前馈与腿长 PID 计算得到的作用力）与期望姿态力矩，精准转化为底层关节电机的执行力矩。本文首先证明了采用特定比例参数的物理腿部机构（缩放五连杆）与标准同轴五连杆机构的等效性，随后基于此等效模型，详细推导了雅可比矩阵及其 VMC 力矩映射关系。

---

## 1. 缩放五连杆与标准五连杆的等效性证明

针对包含平行四边形或内部相似约束的物理腿部构型，当其连杆参数满足特定比例时，可以从运动学与静力学双重维度证明其等价于一个同轴的虚拟标准五连杆机构。这意味着在忽略连杆自身动力学（质量、惯量）的前提下，两者可以完全采用相同的控制器进行控制。

### 1.1 运动学等效证明

已知实际物理连杆的参数比例关系为：
$$k = \frac{l_3}{l_2} = \frac{l_4}{l_1} = 0.54$$
由此可得连杆长度关系为 $l_3 = k l_2$ 且 $l_4 = k l_1$。

建立以电机同轴安装点 $A(E)$ 为原点的复平面坐标系，设左右两电机的输入角度分别为 $\varphi_1$ 和 $\varphi_4$。对于物理机构，其末端点 $C$（轮毂中心）的位置可以由右侧的真实运动支链直接表示：
$$\vec{r}_C = \vec{l}_1 + \vec{l}_2 = l_1 e^{i\varphi_1} + l_2 e^{i\varphi_2}$$

同时，观察左侧驱动支链。由于该腿部机构通过几何相似性构成了缩放映射，点 $C$ 的坐标可以等效为对中间节点 $G$ 的坐标进行 $\frac{1}{k}$ 的放大映射：
$$\vec{r}_C = \frac{1}{k} (\vec{AF} + \vec{FG}) = \frac{1}{k} (l_3 e^{i\varphi_4} + l_4 e^{i\theta_4})$$

将比例关系 $l_3 = k l_2$ 与 $l_4 = k l_1$ 代入上式可得：
$$\vec{r}_C = \frac{1}{k} (k l_2 e^{i\varphi_4} + k l_1 e^{i\theta_4}) = l_2 e^{i\varphi_4} + l_1 e^{i\theta_4}$$

将左右两侧的支链方程联立，得到关于点 $C$ 的闭环矢量方程：
$$l_1 e^{i\varphi_1} + l_2 e^{i\varphi_2} = l_2 e^{i\varphi_4} + l_1 e^{i\theta_4}$$

**证明结论**：该闭环方程正是一个标准的同轴五连杆机构（基座节点均重合于 $A$）的正运动学方程。其中，右侧支链的连杆长度为 $l_1$ 和 $l_2$，而左侧虚拟支链的连杆长度为 $l_2$ 和 $l_1$。这在运动学上充分证明了该物理连杆完全等效于一个交叉对称的虚拟五连杆机构。

### 1.2 静力学等效证明

静力学等效建立在运动学等效的基础之上。根据虚功原理，在一个处于平衡状态的系统中，关节空间与任务空间之间的力学传递由雅可比矩阵（Jacobian Matrix）唯一决定。

设广义关节速度为 $\dot{\mathbf{q}} = [\dot{\varphi}_1, \dot{\varphi}_4]^T$，末端点 $C$ 的线速度为 $\dot{\mathbf{x}}_C = [\dot{x}_C, \dot{y}_C]^T$，两者满足：
$$\dot{\mathbf{x}}_C = \mathbf{J} \dot{\mathbf{q}}$$

由于在 1.1 节中已经得出，该物理连杆机构末端 $C$ 的正运动学解析式 $\vec{r}_C = f(\varphi_1, \varphi_4)$ 与等效虚拟五连杆机构完全相同，因此对其求全微分所得到的雅可比矩阵 $\mathbf{J} = \frac{\partial f}{\partial \mathbf{q}}$ 也必然完全相同。由此推导出的关节力矩 $\mathbf{T}$ 与末端作用力 $\mathbf{F}$ 之间的映射关系 $\mathbf{T} = \mathbf{J}^T \mathbf{F}$ 也是恒等一致的。

**证明结论**：两者具有完全等价的静力学映射关系，因此在底层力控算法（VMC）中，可以直接采用等效标准五连杆的模型进行力矩分配与解算。

## 2. 平面五连杆机构雅可比解析推导与 VMC 力矩映射理论分析

基于上述等效结论，下文针对标准平面五连杆机构展开雅可比矩阵的解析推导及控制映射闭环分析。

### 2.1 机构描述与正运动学建模

在平面五连杆机构中，定义转动副 A 和 E 为主动关节，分别由两台电机驱动，其关节角记为 $\phi_1$ 和 $\phi_4$。控制目标主要关注机构末端点 C 的工作空间状态，可分别用直角坐标 $(x_C, y_C)$ 或极坐标 $(L_0, \phi_0)$ 表示。

基于闭环矢量法，通过五连杆左右两部分列写点 B 和点 D 到点 C 的几何约束，可建立以下方程：
$$\begin{cases} x_B + l_2 \cos \phi_2 = x_D + l_3 \cos \phi_3 \\ y_B + l_2 \sin \phi_2 = y_D + l_3 \sin \phi_3 \end{cases}$$

消去中间变量 $\phi_3$ 后，可求解出被动关节角 $\phi_2$ 随主动关节变化的非线性解析式：
$$\phi_2 = 2 \arctan\left(\frac{B_0 + \sqrt{A_0^2 + B_0^2 - C_0^2}}{A_0 + C_0}\right)$$

进而可得末端点 C 的直角坐标系正运动学映射：
$$\begin{cases} x_C = l_1 \cos \phi_1 + l_2 \cos \phi_2 \\ y_C = l_1 \sin \phi_1 + l_2 \sin \phi_2 \end{cases}$$

### 2.2 雅可比矩阵的解析推导

雅可比矩阵 $J$ 描述了关节空间微小位移到工作空间微小位移的线性映射关系，即 $\delta x = J \delta q$。对于上述包含复杂嵌套和反三角函数的正运动学解析解，直接调用符号运算工具求导会得到极其复杂的结果，难以部署于运算资源受限的嵌入式平台。

为避免计算灾难，本文采用**对位置方程隐式求导提取速度映射系数**的方法。对点 C 的正运动学方程求时间导数：
$$\begin{cases} \dot{x}_C = -l_1 \dot{\phi}_1 \sin \phi_1 - l_2 \dot{\phi}_2 \sin \phi_2 \\ \dot{y}_C = l_1 \dot{\phi}_1 \cos \phi_1 + l_2 \dot{\phi}_2 \cos \phi_2 \end{cases}$$

为消去中间不可测变量 $\dot{\phi}_2$，对闭环约束等式求导得：
$$\begin{cases} \dot{x}_B - l_2 \dot{\phi}_2 \sin \phi_2 = \dot{x}_D - l_3 \dot{\phi}_3 \sin \phi_3 \\ \dot{y}_B + l_2 \dot{\phi}_2 \cos \phi_2 = \dot{y}_D + l_3 \dot{\phi}_3 \cos \phi_3 \end{cases}$$

消去 $\dot{\phi}_3$ 即可解析解出 $\dot{\phi}_2$ 关于已知参量的表达式：
$$\dot{\phi}_2 = \frac{(\dot{x}_D - \dot{x}_B)\cos \phi_3 + (\dot{y}_D - \dot{y}_B)\sin \phi_3}{l_2 \sin(\phi_3 - \phi_2)}$$

将该式代回末端速度方程，并经过三角函数两角和差公式的化简合并，最终可提取出只包含基础三角函数的标准雅可比矩阵形式：
$$\begin{bmatrix} \dot{x}_C \\ \dot{y}_C \end{bmatrix} = J \begin{bmatrix} \dot{\phi}_1 \\ \dot{\phi}_4 \end{bmatrix}$$

这种基于速度映射推导出的 $J$ 有效避免了根号与复杂代数分式，大幅提升了控制周期的实时计算效率。

### 2.3 基于虚功原理的力矩映射

在获取了表征运动微分关系的雅可比矩阵后，需建立静力学层面的力矩映射。根据虚功原理（Principle of Virtual Work），处于平衡状态的系统在任意兼容的微小虚位移下，外力所做总虚功为零。

假设系统输入侧的电机执行力矩向量为 $T = \begin{bmatrix} T_1 & T_2 \end{bmatrix}^T$，产生的微小虚位移为 $\delta q$；工作空间末端输出对抗环境的虚拟力向量为 $F$，末端产生的微小虚位移为 $\delta x$。
* 电机力矩输入的虚功为：$T^T \delta q$
* 末端向环境输出虚拟力的做功为：$(-F)^T \delta x$

由能量守恒建立虚功平衡方程：
$$T^T \delta q + (-F)^T \delta x = 0$$

将微分运动学关系 $\delta x = J \delta q$ 代入上式得：
$$T^T \delta q - F^T (J \delta q) = 0$$
$$(T^T - F^T J) \delta q = 0$$

由于 $\delta q$ 是任意独立的微小虚位移，其系数矩阵必须恒为零，即 $T^T = F^T J$。等式两端取转置，最终推导出静力学映射核心公式：
$$T = J^T F$$

### 2.4 极坐标系下的完整控制映射闭环

在双轮足机器人等实际应用中，内环控制器（如用于腿长控制的 PID 或重力前馈）通常输出的是沿极径方向的期望腿推力 $F$ 和控制姿态的髋关节扭矩 $T_p$。此时需要通过坐标变换矩阵将其转换到直角坐标系下。

首先，利用变换矩阵 $M$ 将极坐标系下的期望力向量映射为末端沿 $L_0$ 及其垂向的虚拟力 $[F_t, F_c]^T$：
$$\begin{bmatrix} F_t \\ F_c \end{bmatrix} = M \begin{bmatrix} F \\ T_p \end{bmatrix}$$

随后，利用旋转矩阵 $R$ 将力向量旋转到全局的 XY 直角坐标系中，得到直角坐标系下的虚拟力 $[F_x, F_y]^T$：
$$\begin{bmatrix} F_x \\ F_y \end{bmatrix} = R \begin{bmatrix} F_c \\ F_t \end{bmatrix}$$

最终，将该期望力代入前述推导的虚功映射方程中，即可获得完整串联的底层控制算法部署级力矩映射公式：
$$\begin{bmatrix} T_1 \\ T_2 \end{bmatrix} = J^T R M \begin{bmatrix} F \\ T_p \end{bmatrix}$$
